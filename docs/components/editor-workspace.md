# 编辑工作台组件说明

## 组件职责

编辑工作台负责公众号文章从输入到导出的完整本地闭环：

- 输入文章元信息。
- 编辑正文富文本。
- 插入常见公众号内容块。
- 插入图片和视频媒体元素。
- 插入六宫格插图媒体块。
- 插入代码块和简单表格。
- 选择已有媒体块并在右侧面板更新内容。
- 选择排版主题、自定义主题色和文章模板。
- 搜索并筛选模板资源库。
- 预览微信图文效果。
- 复制富文本或导出 HTML。
- 支持正文撤销/重做和本地自动保存。

## 模块边界

- `index.html` 提供静态 DOM 容器和 SEO 内容。
- `src/app.js` 维护编辑器状态、模板数据、HTML 转换和复制导出。
- `src/data.js` 维护主题、模板、六宫格默认数据和存储常量。
- `src/utils.js` 维护防抖、转义、URL 清洗和文件名 slug 工具函数。
- `src/styles.css` 负责布局、响应式和预览外观。
- `localStorage` 只保存本机草稿，不承担正式数据存储。

## 数据接口

当前没有远端 API。后续可在以下边界扩展：

- `buildArticleHtml()`：接入云端模板渲染或 AI 排版。
- `themes`：后续可替换为远程主题库，当前内置 12 套首发主题和 1 套自定义主题。
- `templates`：后续可替换为远程模板市场，当前内置 12 个首发模板。
- `renderTemplateLibrary()`：模板搜索、分类筛选和模板卡片渲染入口。
- `insertImageBlock()`：图片元素插入入口，支持远程图片和本地图片 data URL。
- `selectMediaNode()`：编辑区媒体块选中入口。
- `updateSelectedMedia()`：根据当前选中媒体块执行更新。
- `upsertImageBlock()`：未选中时插入图片，选中图片/六宫格卡片时更新图片。
- `videoNodeToWechatHtml()`：视频元素导出入口，输出微信兼容的视频链接卡片。
- `buildSixGridHtml()`：六宫格插图块生成入口。
- `sixGridNodeToWechatHtml()`：六宫格微信兼容导出入口。
- `nodeToWechatHtml()`：代码块、表格、提示块、图片、视频和普通段落的导出分发入口。
- `pushHistory()`：正文编辑历史栈入口。
- `undo()` / `redo()`：正文撤销和重做入口。
- `saveState()`：接入用户账号和云端草稿。

## 交互状态

- 成功：复制、保存、下载后通过状态栏和 toast 提示。
- 空内容：编辑区为空时预览显示“正文待补充”。
- 空搜索：模板资源库会显示 `0 个模板`，不影响当前草稿。
- 图片失败：未填写有效图片链接时给出状态提示。
- 视频限制：视频以链接卡片导出，正式视频发布仍建议在微信后台插入或确认。
- 兼容复制：浏览器不支持 `ClipboardItem` 时回退到选区复制。
- 历史记录：编辑区正文输入、模板切换、清空草稿和媒体更新都会进入撤销栈。
- 自动保存：编辑输入和表单变更通过防抖写入 `localStorage`。
- 移动端：三栏布局折叠为单列。

## 视觉结构

- 桌面端以三栏工作台为主：左侧编辑、中间预览、右侧配置。
- `1260px` 以下右侧配置区先折叠为下方两列，保留编辑和预览并排。
- `920px` 以下切换为单列，左侧编辑、预览、右侧配置按任务顺序排列。
- 模块使用分隔线分组，不在面板内部继续叠加卡片。
- 编辑区、主题库、模板库和右侧面板都设置稳定滚动和固定最小尺寸，降低内容变化造成的布局跳动。

## 可访问性

- 主区域使用 `main`、`aside`、`article` 等语义标签。
- 输入项有明确 `label`。
- 预览尺寸切换、工具栏和操作按钮可键盘访问。
